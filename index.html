Vendor prefixes can be very annoying because of how much code there is for it and having to type it over and over again. Mixins are a great way to use the same block of code multiple times.
Example: @mixin border-radius($radius) {
-webkit-border-radius: $radius;
-moz-border-radius: $radius;
-ms-border-radius: $radius;
border-radius: $radius;
} #mix { @include border-radius(10px); }
